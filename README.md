# PictureSorter
A Python application that sorts your Pictures based on the faces in it.

## Installation

```bash
pipenv install
```

## Usage
1. Train your models
2. file structure
3. Put your files inside the `validation` directory.
4. `python3 main.py`


## Running tests
You can run test via this command:
```bash
python -m unittest discover tests
```
